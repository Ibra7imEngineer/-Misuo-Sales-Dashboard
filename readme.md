---

## 📋 نظرة عامة على المشروع (About)

تحليل بيانات مبيعات تفاعلي تم بناؤه باستخدام **Microsoft Excel** لمحاكاة سيناريو بيزنس لشركة تجزئة تبيع منتجات متعددة (مثل كوتونيل أو العرجاني). يهدف المشروع إلى تحويل البيانات الخام إلى رؤى بصيرة (Insights) قابلة للتنفيذ لمساعدة الإدارة العليا في مراقبة الأداء اللحظي وتحديد فرص النمو.

## 🛠 المهارات والتقنيات المستخدمة (Technical Skills)

- **Data Extraction & Transformation:** استخدام **Power Query** لتنظيف ومعالجة البيانات قبل التحليل (ETL).
- **Data Modeling:** إنشاء نماذج بيانات تعتمد على جداول محورية (Advanced Pivot Tables) لربط مصادر البيانات المختلفة.
- **UI/UX Design:** تصميم واجهة مستخدم احترافية بنظام (Dark Mode) متناسق، مع توزيع استراتيجي للعناصر لضمان سهولة القراءة (Dashboard Design Principles).
- **Advanced Functionality:** ربط الفلاتر (Slicers) باستخدام **Report Connections** لضمان تفاعلية اللوحة بالكامل.

---

## 📈 التحليلات المشمولة (Analysis Covered)

### 1. مؤشرات الأداء الرئيسية (KPIs)

تتبع لحظي لأهم مقاييس الأداء:

- إجمالي المبيعات (Total Sales).
- إجمالي عدد الطلبات (Total Orders).
- متوسط قيمة الطلب (Average Order Value).

### 2. تحليل الأداء التشغيلي والزمني

- **حالة التسليم (Delivery Status):** مراقبة نسب الشحنات المسلمة والمتأخرة والمرفوضة لتحسين الكفاءة اللوجستية.
- **المبيعات حسب المنصة (Sales by Platform):** تحليل أداء قنوات البيع المختلفة (أونلاين، فروع، إلخ).
- **التحليل الزمني:** القدرة على تحليل الأداء شهرياً أو ربع سنوي (Q1, Q2...).

### 3. تحليل المنتجات وسلوك المستهلك

- **المنتجات الأعلى مبيعاً (Best Seller):** ترتيب المنتجات حسب حجم المبيعات.
- **الألوان الأكثر مبيعاً (Best Color sold):** تحليل تفضيلات المستهلكين للألوان.
- **الفلترة الدقيقة:** إمكانية تحليل الأداء حسب اسم المنتج أو نوع العميل (فردي، شركة).

## 💻 كيفية تشغيل المشروع واستخدامه (How to Use)

_(ملاحظة: بما أن جيت هب لا يعرض ملفات Excel بشكل مباشر، يجب تحميل الملف)_

1. قم بتحميل الملف ذو الامتداد `.xlsx` الموجود في هذا المستودع (Repository).
2. افتح الملف باستخدام برنامج **Microsoft Excel** (نسخة 2016 أو أحدث لضمان التوافق).
3. عند فتح الملف، قد تحتاج إلى تفعيل المحتوى (Enable Content) لضمان عمل الفلاتر التفاعلية.
4. استخدم لوحة الفلاتر الموجودة على اليمين/اليسار لتغيير البيانات المعروضة فوراً.
